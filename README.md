# Solid Community Server Article

A systems paper about the [Solid Community Server](https://github.com/solid/community-server)

## Development mode
```
bundle install
bundle exec guard
```

## Build
```
bundle install
bundle exec nanoc compile
```

View on http://localhost:3000/
